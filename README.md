# Reproducibility Artifact

## Description
This artifact reproduces geographic GNSS track visualizations using Python.

## How to Run
1. Launch the artifact on Chameleon Cloud via Trovi
2. Open the notebook in `notebook/analysis.ipynb`
3. Run all cells

## Expected Output
- Geographic track plots rendered inline
- Figures saved under `results/` (if enabled)

## Data
All required CSV files are included in the `data/` directory.

## Environment
Python dependencies are installed automatically from `requirements.txt`.
